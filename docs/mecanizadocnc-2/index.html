3-Mecanizado CNC 2.5

Mecanizado CNC 2.5

En este post enseñaremos a hacer el respaldo de la silla fresia empezada en el post anterior.

Para partir debemos separar el respaldo del resto de la silla para trabajarlo solo, para esto debemos seleccionar el respaldo y a la izquierda hacer clic derecho en bodies->Respaldo y elegir la opción “Create new component from bodies”. Una vez tengamos este nuevo componente este aparecerá abajo a la izquierda como un nuevo componente y le haremos click derecho y lo exportaremos al computador como archi fusión.

FOTO RESPALDO SOLO

Pasos

  1. Crear un nuevo solido que sea un cubo que pueda cubrir por completo el respaldo.
  2. Dibujaremos con la opción Project (P) un rectángulo de las dimensiones del respaldo
  3. Seleccionaremos la opción “adaptive clearing” y elegir nuestra fresa de 6mm
  4. Debemos volver a cambiar de 1mm a 3mm en “Passes” y seleccionar el cuadrado proyectado como “matchmaking boundarie”
  5. Luego del adaptive clearing que es un paso de sacado de material, procedemos con la opción parallel para hacer un acabado mejor
  6. Se deben tener las mismas configuraciones que para el paso de “Adaptive Clearing”
  7. Luego podemos simular ambos pasos juntos para poder ver que funcione correctamente.
