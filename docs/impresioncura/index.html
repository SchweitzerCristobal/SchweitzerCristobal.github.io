Impresion 3D

La impresion 3D nos permite fabricar de manera facil y efectiva muchas cosas y objetos distintos de manera rapida.

Software Cura Cura es el software que utilizaremos para la programacion de la impresion 3D. Este software toma objetos en formato OBJ o STL para transformarlos a capas laminadas, una vez laminado entrega la informacion a la maquina con un lenguaje en codigo para que la impresora entienda las instrucciones.

Cura tiene es el software mas reconocido dado que tiene muchas ventajas, las mas importantes serian que acepta varios tipos de archivo y codifica para varios modelos de impresora y tipos de impresion.

Link de descarga: https://ultimaker.com/es/software/ultimaker-cura

Instrucciones para utilizar Cura

Paso 1: Escoger Impresora - En impresoras no conectadas elegir Creality 3D y elegir Ender-3

Paso 2: Ajustar Configuracion - Elegir el grosor del material 1.75 y el cabezal debe ser de 0,4 mm.

Paso 3: Abrir archivo - Abrir el archivo ya sea formato OBJ o STL

Paso 4 : Slice & Preview - Para terminar seleccionamos slice para que la lamine y luego preview para previsualizar

¿Impresora FDM o SLA?

Por un lado estan las impresoras fdm que son mas duraderas, sencillas y economicas. Al ser las popularizadas de estas es mas facil encontrar mas filamentos y hay variedades mas economicas.

Port otro lado las impresoras LA tienen una mayor precision para imrpimir y maneja materiales mas espesificos como son sus resinas y metales, esto tambien es uno de los factores que aumenta su precio. Ademas el manejo de estas es mas dificil y sin la experiencia podria dañarse rapidamente.
