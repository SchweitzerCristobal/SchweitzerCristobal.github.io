Creación de Molde

Para este post complementario haremos un molde propio en base al video del profesor, mientras destacaremos las partes mas importantes. El tutorial es para moldes de vaciado, por lo que no se aplica para todo tipo de moldes.

Molde de vaciado

Los moldes de vaciado son aquellos que se crean para verter un material en estado liquido como Silicona o Cera entre otros. Estos moldes se usan para la creación de todo tipo de partes y objetos de la vida cotidiana, a continuación, junto al tutorial del molde iremos viendo como se pueden crear fácilmente.

  1. El primer paso para la creación del molde en Fusion será abrir un Sketch y crear una línea de simetría vertical para revolucionar el dibujo que haremos, esta línea se debe dejar como línea de construcción.
  2. Ahora dibujamos una figura como la que se quiera obtener de resultado, no como el resultado del molde, pero del objeto que queremos crear con este, es importante que este no tenga ángulos rectos en las áreas de desmolde porque se puede complicar mucho el desmoldado. Terminamos el Sketch. (A continuacion podemos ver primer un molde con un angulo recto en el extremo que no es el eje, ese molde tendria problemas en el desmoldado, en el segundo esto se soluciona)

  1. Lo siguiente seria Revolucionar (Revolve) el dibujo anterior seleccionando la línea de simetría del paso 1. como eje (Axis)

  1. Ahora tenemos el objeto final, para mejorar aun mas su desmoldado y también obtener un mejor acabado le haremos un fillet en los bordes que van hacia el molde. Segun la seccion del objeto variamos el fillet pero no debe necesariamente ser grande, puede ser bastante pequeño.

  1. Ahora hacemos un segundo sketch desde la cara Top y dibujamos un cuadrado que cubra por completo la figura. Terminamos el Sketch
  2. En este caso extruimos el dibujo (Extrude) para crear un rectángulo que tape la cara superior del objeto, pero que evite tapar por completo la inferior. “El inferior del objeto se entiende como la cara mas plana que no va hacia el molde”

  1. Utilizamos el comando Join entre ambos cuerpos seleccionando primero el paralelepípedo del paso 6. y como herramienta (Tool) el objeto del paso 3. también activamos la opción de “Keep Tools” para no perder el objeto.

  1. Si todo fue creado correcto desde el principio debería estar listo para imprimir, en caso de cualquier error de medida o de creación de las líneas estas se pueden editar y los objetos se alteraran ya que el Sketch esta primero en la línea de tiempo.
